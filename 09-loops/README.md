# For Loops

## What is a Loop?

```js
console.log(1);
console.log(2);
console.log(3);

// this code is not DRY

for (let i = 1; i <= 100; i++) {
  console.log(i);
}
```

## Four Requirements of a For Loop:
1. Sentry
2. Where it starts
3. When it stops
4. How it changes
